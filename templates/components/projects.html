Some of my Projects

e-Commerce Web Application

This project is a Django based eCommerce web application that supports roles for buyers and vendors, whilst also providing stores, products, carts, orders, reviews, REST API endpoints, and X integration. It extends a marketplace-style workflow where vendors can create stores and products, buyers can browse listings and place orders, and selected events can trigger social media posts.

The project includes both traditional Django views and API endpoints for stores and products. It also includes email-based invoice sending during checkout and X/Twitter posting through the Tweepy client.

Technologies used:
  • Python
  • HTML, CSS
  • Django, Django REST Framework
  • Tweepy
  • PyMySQL and mariadb
  • python-dotenv

Exploratory Data Analysis

This project is an exploratory data analysis (EDA) study of an automobile dataset. The notebook focuses on cleaning the raw data, preparing the columns for analysis, and investigating relationships between vehicle price, fuel efficiency, engine size, horsepower, and manufacturer trends.

Conducted end-to-end data preparation and analysis on a vehicle dataset, including data cleaning, preprocessing, and exploratory analysis. Generated insights into vehicle pricing, fuel economy, engine characteristics, and manufacturer trends through statistical analysis and data visualization.

Technologies used:
  • Python
  • Jupyter Notebook
  • Pandas
  • NumPy
  • MatPlotLib
  • Seaborn

Sentiment Analysis: Amazon Product Reviews

This project performs sentiment analysis on Amazon product reviews using Python natural language processing libraries. The main script allows a user to load a review dataset, preprocess the review text, remove stop words, analyse the sentiment of individual reviews, and compare the similarity between two reviews.

The project is implemented as a menu-driven command-line application and is designed to demonstrate practical NLP preprocessing and review analysis workflows.

Technologies used:
  • Python
  • Pandas
  • SpaCy
  • TextBlob
